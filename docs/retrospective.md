# Reflexión final

## Qué aprendí

Durante este proyecto aprendí a integrar un frontend desarrollado con React y TypeScript con un backend construido utilizando Node.js y Express.

También aprendí a diseñar APIs REST, crear componentes reutilizables y gestionar estado global mediante Context API.

## Principales dificultades

Las principales dificultades aparecieron durante:

* Configuración inicial del proyecto.
* Integración entre frontend y backend.
* Gestión de tipos en TypeScript.
* Organización de componentes y rutas.

## Cómo se resolvieron

Los problemas se resolvieron mediante pruebas incrementales, revisión de errores de compilación y validación constante de cada funcionalidad antes de continuar con la siguiente.

## Uso de IA

La inteligencia artificial se utilizó como herramienta de apoyo para:

* Resolver dudas técnicas.
* Explicar conceptos.
* Detectar errores.
* Proponer estructuras de código.
* Ayudar en la documentación.

Todas las decisiones finales fueron revisadas y aplicadas manualmente durante el desarrollo.

## Conclusión

El proyecto permitió conectar conocimientos de frontend, backend, APIs, TypeScript y metodologías ágiles en una aplicación completa.

Además, proporcionó experiencia práctica en la construcción de aplicaciones fullstack modernas.
