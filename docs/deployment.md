# Despliegue

## Frontend

El frontend fue desarrollado utilizando React, TypeScript y Vite.

La aplicación está preparada para desplegarse en Vercel.

### Proceso

1. Conectar el repositorio de GitHub con Vercel.
2. Seleccionar la carpeta del frontend.
3. Ejecutar el proceso de build.
4. Publicar la aplicación.

## Backend

El backend fue desarrollado utilizando Node.js y Express.

Puede desplegarse en plataformas como Render o Vercel.

### Proceso

1. Conectar el repositorio con la plataforma de despliegue.
2. Configurar el comando de inicio.
3. Configurar variables de entorno si son necesarias.
4. Publicar la API.

## Verificaciones realizadas

* Funcionamiento de rutas.
* Comunicación frontend-backend.
* Operaciones CRUD.
* Navegación entre páginas.

## Resultado

La aplicación quedó preparada para ser utilizada en un entorno de producción.
