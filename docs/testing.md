# Testing

## Pruebas realizadas

### Navegación

Se verificó el funcionamiento correcto de todas las rutas de React Router.

### Creación de juegos

Se comprobó que los formularios permiten crear nuevos juegos correctamente.

### Listado

Se verificó que los juegos creados aparecen en la lista.

### Actualización

Se comprobó que el cambio de estado mediante PATCH funciona correctamente.

### Eliminación

Se verificó que los juegos pueden eliminarse correctamente mediante DELETE.

### Estadísticas

Se comprobó que las estadísticas se actualizan automáticamente cuando cambian los datos.

### Backend

Se probaron todos los endpoints REST mediante el frontend y verificando las respuestas obtenidas.

### Responsive

Se realizaron pruebas básicas de visualización en diferentes tamaños de ventana.

## Resultado

Todas las funcionalidades principales funcionan correctamente.
