# Routing

La navegación se implementó utilizando React Router.

## Rutas disponibles

### /

Dashboard principal.

### /games

Listado completo de juegos.

### /add-game

Formulario para crear nuevos juegos.

### /stats

Estadísticas de la colección de juegos.

### /about

Información sobre el proyecto.

### *

Página 404 para rutas inexistentes.

## Navegación

La navegación se realiza mediante un componente Navbar que utiliza enlaces de React Router para evitar recargas completas de página.

## Beneficios

* Navegación rápida.
* Mejor experiencia de usuario.
* Separación clara entre páginas.
