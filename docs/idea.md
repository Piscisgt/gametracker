# Idea del proyecto

## Nombre

GameTracker

## Descripción

GameTracker es una aplicación web que permite gestionar una colección personal de videojuegos y controlar el progreso de cada uno.

## Problema que resuelve

Muchos jugadores tienen grandes bibliotecas de juegos y pierden el control sobre cuáles han terminado, cuáles están jugando actualmente o cuáles desean jugar en el futuro.

## Usuario objetivo

Jugadores de PC, PlayStation, Xbox y Nintendo que desean organizar su biblioteca personal.

## Funcionalidades principales

* Añadir videojuegos.
* Editar videojuegos.
* Eliminar videojuegos.
* Cambiar estado de progreso.
* Buscar videojuegos.
* Filtrar videojuegos.
* Ver estadísticas generales.

## Estados disponibles

* Pendiente
* Jugando
* Completado
* Abandonado

## Funcionalidades opcionales

* Sistema de puntuación personal.
* Categorías por género.
* Portadas de videojuegos.
* Favoritos.

## Mejoras futuras

* Integración con APIs de videojuegos.
* Sistema de usuarios.
* Base de datos persistente.
* Estadísticas avanzadas.
